ReleaseKit 🚀

ReleaseKit is a suite of tools which simplify and improve the release process for shortcut developers and create a more robust and feature-rich experience for end-users.

☁ miCloud

miCloud is a storage solution and version control system for Shortcuts which serves as the underlying structure and foundation for ReleaseKit. miCloud groups shortcuts into two categories, 'Builds' and 'Releases'. Each Build and Release is subgrouped into 'Major Versions', then further subgrouped into 'Minor Versions'. miCloud creates a folder for each shortcut saved. A dictionary including details of the shortcut and also the user is also created. Additionally, all data for a shortcut is stored in its respective folder. All shortcuts included in ReleaseKit pull data from miCloud.

üöÄ Release Notes

Release Notes is, as the name suggests, for creating release notes. Release Notes are documents which accompany the release of a version of software. Release notes help to inform end-users and keep them aware of any changes in a new version of a shortcut. Release notes typically include new features, bug fixes, and enhancements/improvements. In addition to these categories, Release Notes also includes sections for additional resources and general notes.

👾 Avatars

Developers can now create their own avatars with ReleaseKit. There are a few different styles to choose from, allowing everyone to create avatars which are inclusive and which represent them! The avatars created are tied to the user and can be used anywhere-not just in Shortcuts!
